# Manual — Wordlist Generator

เครื่องมือสร้าง Wordlist สำหรับ Pentest — Developed by **penthammachat**

---

## วิธีใช้งาน

1. เปิดไฟล์ `index.html` ในเว็บเบราว์เซอร์ (Chrome, Firefox, Edge)
2. ตั้งค่าเงื่อนไขใน Step 1
3. กด "ถัดไป" เพื่อเริ่มสร้างคำ
4. เมื่อเสร็จแล้ว กด "ถัดไป" อีกครั้งเพื่อดาวน์โหลด

> ไม่ต้อง install อะไรเพิ่ม ไม่ต้องมี server — เปิดไฟล์ตรงๆ ได้เลย

## Step 1: ตั้งค่าเงื่อนไข

### ช่วงความยาว

| Field | คำอธิบาย | ตัวอย่าง |
|-------|----------|----------|
| **ขั้นต่ำ** | จำนวนตัวอักษรน้อยที่สุดของแต่ละคำ | `1` = สร้างคำยาว 1 ตัวอักษรขึ้นไป |
| **สูงสุด** | จำนวนตัวอักษรมากที่สุดของแต่ละคำ | `8` = สร้างคำยาวสุด 8 ตัวอักษร |

**ตัวอย่าง:** ขั้นต่ำ `3` สูงสุด `5` จะสร้างคำยาว 3, 4, และ 5 ตัวอักษร

> ถ้าขั้นต่ำ > สูงสุด ระบบจะแจ้งเตือน

---

### ตัวอักษรที่ต้องการ (Charset)

เลือกกลุ่มตัวอักษรที่ต้องการใช้สร้างคำ:

| Checkbox | ตัวอักษร | ตัวอย่าง |
|----------|----------|----------|
| **A-Z** | ตัวพิมพ์ใหญ่ 26 ตัว | `A B C D ... Z` |
| **a-z** | ตัวพิมพ์เล็ก 26 ตัว | `a b c d ... z` |
| **0-9** | ตัวเลข 10 ตัว | `0 1 2 3 ... 9` |
| **!@#$** | อักขระพิเศษ 30 ตัว | `! @ # $ % ^ & * ( ) _ + - = [ ] { } | ; : , . < > ?` |

**วิธีใช้:**
- เลือกอย่างน้อย 1 กลุ่ม หรือใส่คำกำหนดเอง
- ด้านล่างจะแสดง **Charset Preview** ว่ามีตัวอักษรอะไรบ้างและกี่ตัว

**ตัวอย่างผลลัพธ์:**
- เลือกเฉพาะ `A-Z` + `0-9` → จะสร้างคำจาก `ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789` (36 ตัว)
- ถ้าขั้นต่ำ `2` สูงสุด `3` จะสร้างคำทั้งหมด 36² + 36³ = 1,296 + 46,656 = **47,952 คำ**

> **คำเตือน:** ยิ่งเลือก charset มาก + ความยาวสูงสุดมาก = จำนวนคำจะเพิ่มแบบ exponential เช่น ถ้าเลือกครบทุกกลุ่ม (92 ตัว) แล้วตั้งสูงสุด 8 จะได้คำมากกว่า 100 ล้านคำ

---

### คำกำหนดเอง (Custom Words)

| Field | คำอธิบาย | ตัวอย่าง |
|-------|----------|----------|
| **คำที่ต้องการ** | ใส่คำที่ต้องการใช้ตรงๆ (ไม่สลับตำแหน่ง) | `password, admin, letmein` |
| **คำภาษาไทย** | ใส่คำไทย จะแปลงเป็นแป้นพิมพ์อังกฤษอัตโนมัติ | `ธรรมชาติ, สวัสดี` |

**วิธีใช้:**
- คั่นแต่ละคำด้วย comma (,)
- ระบบจะใช้คำที่ใส่ **ตรงๆ ไม่สลับตำแหน่ง** ไม่สร้าง permutation
- ถ้าเปิด Leetspeak จะสร้าง leet variations เพิ่มจากคำนั้น

**ตัวอย่างผลลัพธ์:**
- คำว่า `password` (เปิด Leetspeak) → `password`, `p@ssw0rd`, `p@ssw0r7` ฯลฯ
- คำว่า `admin` → `admin` (คำเดียว ไม่สลับ)

**คำภาษาไทย:**
- คำว่า `สวัสดี` จะแปลงเป็น `l;,d` ตาม layout Kedmanee
- ถ้าเปิด Leetspeak จะสร้างเพิ่ม เช่น `l;,@d`, `l;,d` เป็นต้น

### Leetspeak

| Checkbox | คำอธิบาย |
|----------|----------|
| **เปิดใช้ Leetspeak** | สร้างคำแบบleet แทนที่ตัวอักษร |

**Mapping ที่ใช้:**

| ตัวอักษร | แทนที่ด้วย |
|----------|-----------|
| a | @ |
| e | 3 |
| i | 1 |
| o | 0 |
| s | 5 |
| t | 7 |
| l | 1 |
| b | 8 |
| g | 9 |

**ตัวอย่างผลลัพธ์:**
- คำว่า `password` → `p@ssw0rd`, `p@ssw0r7`, `pa$$w0rd`, `p@55w0rd` ฯลฯ
- ระบุจะสร้างทุก combination ของการแทนที่ เช่น ถ้าคำมี 4 ตัวที่แทนได้ จะได้ 2⁴ = 16 รูปแบบ

### Prefix / Suffix

| Field | คำอธิบาย | ตัวอย่าง |
|-------|----------|----------|
| **Prefix (ต่อหน้า)** | ใส่ข้อความที่ต้องการต่อหน้าทุกคำ | `admin` |
| **Suffix (ต่อท้าย)** | ใส่ข้อความที่ต้องการต่อท้ายทุกคำ | `123` |

**ตัวอย่างผลลัพธ์:**
- คำ `pass` + prefix `admin` → `adminpass`
- คำ `pass` + suffix `123` → `pass123`
- คำ `pass` + prefix `admin` + suffix `123` → `adminpass`, `adminpass123`

---

### ตัวเลขต่อท้าย (Number Suffix)

| ตัวเลือก | คำอธิบาย | ตัวอย่าง |
|----------|----------|----------|
| **ไม่ต่อ** | ไม่เพิ่มตัวเลข | `pass` |
| **0-9** | ต่อเลข 0 ถึง 9 | `pass0, pass1, ... pass9` |
| **00-99** | ต่อเลข 00 ถึง 99 | `pass00, pass01, ... pass99` |
| **000-999** | ต่อเลข 000 ถึง 999 | `pass000, pass001, ... pass999` |

**ตัวอย่างผลลัพธ์:**
- คำ `admin` + `00-99` → `admin00, admin01, admin02, ... admin99` (รวม 100 คำ)

---

### Year / ปี

| Field | คำอธิบาย | ตัวอย่าง |
|-------|----------|----------|
| **จากปี** | ปีเริ่มต้น | `2020` |
| **ถึงปี** | ปีสิ้นสุด | `2026` |
| **ต่อปีท้ายคำ** | Checkbox เปิด/ปิด | ✓ |

**ตัวอย่างผลลัพธ์:**
- คำ `password` + ปี 2020-2026 → `password2020, password2021, ... password2026` (7 คำ)

> **หมายเหตุ:** ถ้ามี prefix/suffix/number suffix ด้วย ระบบจะคำนวณทับซ้อนกัน ทำให้จำนวนคำเพิ่มขึ้นมาก

### Combine Words

| Field | คำอธิบาย | ตัวอย่าง |
|-------|----------|----------|
| **คำที่ต้องการ combine** | ใส่หลายคำคั่นด้วย comma | `admin, root, password` |
| **Separator** | ตัวคั่นระหว่างคำ | `-`, `_`, `.`, หรือไม่มี |

**ตัวเลือก Separator:**

| ตัวเลือก | ผลลัพธ์ |
|----------|---------|
| ไม่มี | `adminroot, adminpassword, rootadmin, rootpassword, ...` |
| Dash | `admin-root, admin-password, root-admin, ...` |
| Underscore | `admin_root, admin_password, root_admin, ...` |
| Dot | `admin.root, admin.password, root.admin, ...` |

**ตัวอย่าง:** คำ `admin, root` + separator `-` → `admin-root, root-admin`

### Import Wordlist

| Field | คำอธิบาย |
|-------|----------|
| **อัปโหลดไฟล์ .txt** | นำเข้า wordlist ที่มีอยู่แล้ว |

**วิธีใช้:**
1. สร้างไฟล์ `.txt` โดยใส่ 1 คำต่อบรรทัด

3. ระบบจะแสดงจำนวนคำที่โหลดไว้

**ตัวอย่างไฟล์ input:**
```
password
admin123
letmein
welcome1
p@ssw0rd
```

> คำที่ import มาจะถูก merge เข้ากับคำที่สร้าง และ deduplicate อัตโนมัติ

## Step 3: ดาวน์โหลด

### ตัวอย่างคำ

แสดง **50 คำแรก** และ **50 คำสุดท้าย** จาก wordlist เพื่อให้ตรวจสอบก่อนดาวน์โหลด

### ตัวเลือกดาวน์โหลด

| Field | คำอธิบาย | ตัวอย่าง |
|-------|----------|----------|
| **ชื่อไฟล์** | ชื่อไฟล์ output | `wordlist` |
| **Export Format** | รูปแบบไฟล์ | ดูด้านล่าง |
| **แยกไฟล์** | วิธีแยกไฟล์ | ดูด้านล่าง |
| **ดาวน์โหลดเป็น .zip** | รวมไฟล์เป็น zip (ถ้าแยกหลายไฟล์) | ✓ |

---

### Export Format

| ตัวเลือก | นามสกุล | คำอธิบาย | ตัวอย่าง |
|----------|---------|----------|----------|
| **Plain Text** | `.txt` | ไฟล์ข้อความธรรมดา 1 คำต่อบรรทัด | `password`<br>`123456`<br>`qwerty` |
| **Hashcat Mask** | `.hc` | รูปแบบสำหรับ hashcat | ใช้กับ `hashcat -a 3` |
| **John the Ripper** | `.john` | รูปแบบสำหรับ john | ใช้กับ `--format=` |

**วิธีใช้กับเครื่องมือ:**
```bash
# Hashcat
hashcat -a 0 -m 0 hash.txt wordlist.txt

# John the Ripper
john --wordlist=wordlist.txt hash.txt
```

---

### แยกไฟล์ (Split)

| ตัวเลือก | คำอธิบาย | ตัวอย่าง |
|----------|----------|----------|
| **ไม่แยก** | ไฟล์เดียวทั้งหมด | `wordlist.txt` |
| **แยกตามจำนวนไฟล์** | กำหนดจำนวนไฟล์ที่ต้องการ | 3 ไฟล์ → `wordlist_001.txt`, `_002.txt`, `_003.txt` |
| **แยกตามขนาดไฟล์** | กำหนดขนาดสูงสุดต่อไฟล์ (MB) | 5MB → ไฟล์ละไม่เกิน 5MB |

**ตัวอย่างแยกตามจำนวน:**
- คำ 30,000 คำ + แยก 3 ไฟล์ → ไฟล์ละ 10,000 คำ

**ตัวอย่างแยกตามขนาด:**
- คำ 100,000 คำ + แยกไฟล์ละ 5MB → ได้หลายไฟล์ ขึ้นอยู่กับความยาวเฉลี่ยของคำ

> ชื่อไฟล์จะ auto-number: `wordlist_001.txt`, `wordlist_002.txt`, ...

## ตัวอย่างการใช้งานจริง

### ตัวอย่าง 1: สร้าง wordlist สำหรับ password ทั่วไป

```
ขั้นต่ำ: 6
สูงสุด: 8
Charset: A-Z + a-z + 0-9
```
→ ได้คำหลายล้านคำ ครอบคลุม password ความยาว 6-8 ตัวอักษร

### ตัวอย่าง 2: สร้าง wordlist จากคำที่รู้

```
คำที่ต้องการ: password, admin, letmein
Leetspeak: ✓
ตัวเลขต่อท้าย: 0-9
```
→ ได้คำเช่น `password`, `p@ssw0rd`, `password0`, `password1`, ... `admin`, `admin0`, ... `letmein`, `letme1n`, `letmein0`, ...

> หมายเหตุ: คำจะถูกใช้ตรงๆ ไม่สลับตำแหน่ง ถ้าต้องการ permutation ให้ใส่ในหลายคำ เช่น `man, mna, amn`

### ตัวอย่าง 3: สร้าง wordlist จากคำไทย

```
คำภาษาไทย: สวัสดี, ธรรมชาติ
Leetspeak: ✓
```
→ ได้คำเช่น `l;,d`, `l;,@d`, `Tii,=k9B` ฯลฯ

### ตัวอย่าง 4: สร้าง wordlist แบบ combine

```
คำ combine: admin, root, pass
Separator: -
Suffix: 123
Year: 2024-2026
```
→ ได้คำเช่น `admin-root1232024`, `pass-admin1232025` ฯลฯ

## หมายเหตุทางเทคนิค

- **Deduplication:** คำซ้ำจะถูกตัดออกอัตโนมัติ ไม่ต้องกังวลเรื่องคำซ้ำ
- **Web Worker:** การสร้างคำทำงานบน thread แยก ไม่ทำให้ browser ค้าง พร้อม progress bar real-time
- **Custom Words:** คำที่ใส่จะถูกใช้ตรงๆ ไม่สลับตำแหน่ง ไม่สร้าง permutation
- **Size Warning:** ถ้าขนาด output เกิน 10MB จะแสดงคำเตือนให้ใช้ Split
- **File Naming:** ไฟล์ที่แยกจะ auto-number ด้วย leading zeros (001, 002, 003)

---

## ข้อจำกัด

- ไม่รองรับการ brute-force แบบ mask (เช่น `???9` สำหรับ 4 ตัวอักษร + เลขท้าย)
- ไม่รองรับ rule-based mutation แบบ hashcat (เช่น `Rule: $1 $2 $3`)
- ไม่รองรับการ import wordlist แบบ hashcat rule file
- ทุกอย่างทำงานบน client-side ไม่มี server-side processing
